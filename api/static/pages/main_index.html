Deloton Ride Data API

/ride

Retrieve information about rides.

/ride/:id

GET a ride with a specific ID.

Query Parameters:

  • expanded=true - Fetch all readings associated with ride.
  • summary=true - Summarise all readings associated with ride.

DELETE a ride with a specific ID.

/rider

Retrieve information about riders.

/rider/:user_id

GET rider information (e.g., name, gender, age, avg. heart rate, number of rides).

/rider/:user_id/rides

GET all rides for a rider with a specific ID.

Query Parameters:

  • expanded=true - Fetch all readings associated with each ride.
  • summary=true - Summarise all readings associated with each ride.

/daily

GET all rides in the current day.

Query Parameters:

  • expanded=true - Fetch all readings associated with each ride.
  • summary=true - Summarise all readings associated with each ride.
  • date=01-01-2020 - Get all rides for a specific date.
